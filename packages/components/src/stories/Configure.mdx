# Vue3 ProComponents 文档

## 动机

基于 `element-plus` 组件库，仿照 Antd 的 ProComponent 组件库，实现了一套 Vue3 的 ProComponent 组件库，大部分逻辑通过 json 配置的方式即可实现，简化业务开发的成本。

## 使用

### 安装

```json
npm install @df/toco-ui-vue --save

```

### 引用

```ts
import ProTable, { ProForm } from '@df/toco-ui-vue'
```

具体用法参考 [ProTable](/docs/example-protable--docs) 和 [ProForm](/docs/example-proform--docs) 的文档。
